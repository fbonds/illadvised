## App Store Preview Video Guidelines

Apple's requirements for app preview videos:
- **Length**: 15-30 seconds (recommended)
- **Orientation**: Portrait (for iPhone apps)
- **Resolution**: 1080p or higher
- **Format**: .mov or .mp4
- **Audio**: Optional but recommended

### Editing Tips

To create App Store preview from this video:
1. Extract the most compelling 15-30 seconds
2. Focus on:
   - The unique dual-camera split screen
   - Live sensor data overlay
   - The "Ill Advised" branding
   - Recording button interaction
3. Add text overlays if needed:
   - "Record your phone's final moments"
   - "Dual camera + sensor data"
   - "Stream to social media"
4. Consider adding upbeat/ironic music
5. End with app icon and name
